Passo 1
Parità della funzione

Calcola f(−x) e confrontala con f(x).

Funzione PARI
f(−x) = f(x)

Simmetrica rispetto all'asse y.
Basta studiare x ≥ 0 e specchiare.

simmetria rispetto a y
Funzione DISPARI
f(−x) = −f(x)

Simmetrica rispetto all'origine.
Basta studiare x ≥ 0 e ruotare di 180°.

simmetria rispetto all'origine

Se f(−x) ≠ f(x) e f(−x) ≠ −f(x) → funzione né pari né dispari. Nessuna simmetria.

Dominio — Campo di Esistenza (C.E.)
Frazione
B(x) ≠ 0

Il denominatore non può essere zero. Risolvi B(x) = 0 e escludi quei valori.

Radice pari (√)
A(x) ≥ 0

Il contenuto deve essere ≥ 0.
Radice pari al denominatore: A(x) > 0 (stretto).

Radice dispari (∛)
x ∈ ℝ

Nessun limite. Esiste per tutti i reali.
Radice dispari al denominatore: A(x) ≠ 0.

Logaritmo
f(x) > 0

L'argomento del logaritmo deve essere strettamente positivo.

Esponenziale a^f(x)
Dominio di f(x)

Calcola il dominio dell'esponente f(x). La base a > 0 e a ≠ 1.

sin / cos
x ∈ ℝ

Seno e coseno esistono per tutti i reali.

tan(x)
x ≠ π/2 + kπ

Escludi i valori dove il coseno è zero: x ≠ π/2 + kπ, k ∈ ℤ

  • Ricordati di tenere conto del C.E.: fuori dal dominio non puoi scrivere nulla
  • Per frazioni: studia il segno di numeratore e denominatore separatamente, poi applica la regola dei segni
Regola dei segni
Frazione N/D

+ / + = +    − / − = +
+ / − = −    − / + = −

Fai la tabella degli zeri di N e D.

Risultato sul grafico
Segno positivo → curva sopra asse x

Segnati gli intervalli dove f > 0 direttamente sul grafico con tratti sopra l'asse.

Passo 4
Intersezioni con gli assi
Asse x (ascisse)
f(x) = 0

Risolvi l'equazione f(x) = 0.
Ogni soluzione è un punto della forma (x₀, 0).
Deve stare nel dominio!

Asse y (ordinate)
f(0) = ?

Riporta i punti trovati subito sul grafico. Sono dati certi.

Passo 5
Limiti
Quando calcolare i limiti
  • Limiti a +∞ e −∞ — sempre, se il C.E. arriva all'infinito
  • Limiti nei punti esclusi dal dominio — sia da destra (x→c⁺) che da sinistra (x→c⁻)
Forme indeterminate
Quando il limite non è immediato
  • ∞/∞ — raccogli il termine di grado maggiore
  • 0/0 — semplifica / fattorizza
  • ∞ − ∞ — porta a forma comune
  • 0 · ∞ — riscrivi come 0/0 o ∞/∞
  • 1^∞ → formula speciale con e
Limite razionale a ∞
Dividi per x^n (grado massimo)

Raccogli al numeratore e denominatore il termine con la x di grado più alto. Tutti gli altri → 0.

lim (3x²+x)/(x²−5) = 3/1 = 3
→ asintoto orizzontale y = 3
Limite da destra/sinistra
Nei punti di "buco" nel dominio

Se c ∉ dominio (es. denominatore = 0):

limx→c⁺ f(x) = ?
limx→c⁻ f(x) = ?

I segni determinano se la curva va a +∞ o −∞.

Limiti notevoli utili
Da ricordare
limx→0 sin(x)/x = 1
limx→∞ (1+1/x)ˣ = e
limx→0 (1+x)^(1/x) = e
Passo 6
Asintoti
Asintoto Orizzontale
y = c    equazione: retta orizzontale
limx→+∞ f(x) = L   →   asintoto orizzontale y = L (a +∞)
limx→−∞ f(x) = M   →   asintoto orizzontale y = M (a −∞)

Se il limite è un numero finito → asintoto. Se è ∞ → niente asintoto orizzontale lì.

la curva si avvicina all'asintoto orizzontale senza mai toccarlo (o toccandolo)
Asintoto Verticale
x = c    equazione: retta verticale
limx→c⁺ f(x) = ±∞   oppure   limx→c⁻ f(x) = ±∞

Basta che uno dei due limiti laterali sia infinito. Si trova nei punti esclusi dal dominio.

la curva va all'infinito avvicinandosi a x = c
Asintoto Obliquo
y = mx + q    retta inclinata

Esiste solo se il limite a ∞ NON è finito e NON è ∞ (cioè, il grado del numeratore = grado denominatore + 1).

m = limx→∞ f(x)/x

q = limx→∞ [f(x) − mx]
  • Se m = 0 → non è obliquo ma orizzontale
  • Se m = ∞ → non esiste
  • Calcola m prima, poi q con m già trovato

Derivata seconda — concavità

Capitolo 2
Punti di discontinuità — 3 specie

Un punto x = c è di discontinuità quando la funzione non è continua in quel punto (perché non è definita lì, o il limite non esiste, o non coincide con il valore della funzione).

Prima specie
Discontinuità a salto
Il limite destro e il limite sinistro esistono entrambi e sono finiti, ma sono diversi tra loro.
limx→c⁻ f(x) = L₁    limx→c⁺ f(x) = L₂    con L₁ ≠ L₂
la curva "salta" — L₁ ≠ L₂

Il salto vale |L₁ − L₂|. La funzione può o non può essere definita in c — non cambia la specie.

Seconda specie
Discontinuità essenziale
Almeno uno dei due limiti laterali non esiste oppure è infinito.
limx→c⁻ f(x) = ±∞   oppure   limx→c⁺ f(x) = ±∞
 
oppure: almeno uno dei limiti laterali non esiste (es. oscillazione)
almeno un lato va a ∞ — asintoto verticale

Questa specie include i punti dove c'è un asintoto verticale. Ma anche se la funzione oscilla senza limite (come sin(1/x) per x→0) è seconda specie.

Terza specie
Discontinuità eliminabile
Il limite destro e il limite sinistro esistono, coincidono (il limite esiste), ma il valore della funzione in quel punto è diverso dal limite (o la funzione non è definita lì).
limx→c f(x) = L    ma    f(c) ≠ L    (oppure f(c) non esiste)
"buco" nella curva — il limite esiste ma f(c) è altrove (o mancante)

Si chiama "eliminabile" perché basta ridefinire f(c) = L per rendere la funzione continua. È il tipo più "innocuo".

Riepilogo rapido
Specie Condizione Eliminabile?
I — Salto lim⁻ e lim⁺ esistono finiti ma ≠ No
II — Essenziale Almeno un lim laterale = ∞ o non esiste No
III — Eliminabile lim esiste finito, ma f(c) ≠ lim Sì
Capitolo 3
Funzioni esponenziali e logaritmiche
f(x) = aˣ   (con a > 0, a ≠ 1)
a > 1 (es. 2ˣ, eˣ, 10ˣ)
Esponenziale crescente
  • Passa sempre per (0, 1)
  • x→+∞ : f→+∞
  • x→−∞ : f→0 (asintoto y = 0)
  • Sempre > 0, mai tocca asse x
  • Dominio: ℝ  |  Codominio: (0, +∞)
0 < a < 1 (es. (1/2)ˣ)
Esponenziale decrescente
  • Passa sempre per (0, 1)
  • x→+∞ : f→0 (asintoto y = 0)
  • x→−∞ : f→+∞
  • Sempre > 0, mai tocca asse x
  • È il "ribaltamento" orizzontale di a > 1
Proprietà algebriche — calcolo
Regole dell'esponenziale
aˣ · aʸ = aˣ⁺ʸ
aˣ / aʸ = aˣ⁻ʸ
(aˣ)ʸ = aˣʸ
a⁰ = 1 (per ogni a ≠ 0)
a⁻ˣ = 1/aˣ
Varianti grafiche
f(x) = aˣ + k
Traslazione verticale

Il grafico si sposta di k in su (k > 0) o in giù (k < 0).
Asintoto diventa y = k.

f(x) = aˣ⁺ʰ
Traslazione orizzontale

Il grafico si sposta di h a sinistra (h > 0) o a destra (h < 0).
Il punto (0,1) diventa (−h, 1).

f(x) = −aˣ
Ribaltamento verticale

Il grafico si ribalta sotto l'asse x.
Sempre < 0. Asintoto y = 0 (da sotto).

f(x) = a⁻ˣ
= (1/a)ˣ — decrescente

Riflessione rispetto all'asse y del grafico di aˣ.
Equivalente a usare base 1/a.

f(x) = log_a(x)   (con a > 0, a ≠ 1, x > 0)

Il logaritmo è la funzione inversa dell'esponenziale.
log_a(x) = y  ⟺  aʸ = x

a > 1 (es. log₂, log₁₀, ln)
Logaritmo crescente
  • Passa per (1, 0) sempre (log_a(1) = 0)
  • Passa per (a, 1)
  • x→0⁺ : f→−∞ (asintoto x = 0)
  • x→+∞ : f→+∞ (lentamente)
  • Dominio: (0, +∞)  |  Codominio: ℝ
0 < a < 1
Logaritmo decrescente
  • Passa per (1, 0) sempre
  • x→0⁺ : f→+∞
  • x→+∞ : f→−∞
  • Ribaltamento verticale del caso a > 1
Proprietà algebriche — calcolo
Regole del logaritmo
log_a(M·N) = log_a(M) + log_a(N)
log_a(M/N) = log_a(M) − log_a(N)
log_a(Mⁿ) = n · log_a(M)
log_a(a) = 1
log_a(1) = 0
log_a(aˣ) = x     a^(log_a(x)) = x

Cambio base: log_a(x) = log_b(x) / log_b(a)
Dominio — attenzione!
L'argomento deve essere > 0

Se hai log_a(g(x)), il C.E. è g(x) > 0. Questo è spesso la parte più complicata dello studio.

Es: log(x²−4) → C.E.: x²−4 > 0 → |x| > 2 → x < −2 oppure x > 2

Varianti grafiche
f(x) = log_a(x) + k
Traslazione verticale

Il grafico si sposta di k in su/giù.
Il punto (1, 0) diventa (1, k).
Asintoto verticale rimane x = 0.

f(x) = log_a(x − h)
Traslazione orizzontale

Il grafico si sposta di h a destra (h > 0).
Asintoto diventa x = h.
Dominio diventa x > h.

f(x) = log_a(−x)
Riflessione sull'asse y

Funzione definita solo per x < 0.
Dominio: (−∞, 0). Asintoto: x = 0.

f(x) = |log_a(x)|
Valore assoluto

La parte negativa del grafico si ribalta sopra l'asse x.
Dove log < 0 (per 0 < x < 1 con a > 1) → si specchia.

Esponenziale aˣ
Punti fissi del grafico
  • Passa per (0, 1) sempre
  • Asintoto orizzontale: y = 0
  • Mai tocca asse x
  • Dominio: ℝ
Logaritmo log_a(x)
Punti fissi del grafico
  • Passa per (1, 0) sempre
  • Asintoto verticale: x = 0
  • Mai tocca asse y
  • Dominio: (0, +∞)
Simmetria tra esponenziale e logaritmo

I grafici di f(x) = aˣ e f(x) = log_a(x) sono simmetrici rispetto alla retta y = x (sono funzioni inverse).

aˣ e log_a(x) sono specchiati rispetto a y = x
Trucchi e regole veloci
Come riconoscere il comportamento
  • Se a > 1 → sia esponenziale che logaritmo sono crescenti
  • Se 0 < a < 1 → entrambi decrescenti
  • Esponenziale cresce/decresce rapidamente, logaritmo lentamente
  • Per le equazioni: aˣ = aʸ ⟺ x = y    log_a(x) = log_a(y) ⟺ x = y
  • Disequazione con a > 1: aˣ > aʸ ⟺ x > y
  • Disequazione con 0 < a < 1: aˣ > aʸ ⟺ x < y (senso cambia!)
Errori classici
  • log_a(x + y) ≠ log_a(x) + log_a(y)   (non si può spezzare la somma dentro il log)
  • log_a(x · y) = log_a(x) + log_a(y) ✓   (qui si può, è il prodotto)
  • Disequazione log: se a < 1, il senso della disuguaglianza si inverte
  • Il dominio del log è strettamente > 0, non ≥ 0
